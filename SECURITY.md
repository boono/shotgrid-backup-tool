# Security

## 凭据

不要在 issue、日志、截图或配置文件中提交 ShotGrid API Key。若 Key 曾发送到聊天、工单或公开位置，请立即在 ShotGrid 中轮换。

图形应用会在预检查后清空输入框，并使用短时、单次内存 handle 启动备份；本工具不会主动把 API Key 写入快照。浏览器、操作系统和第三方诊断工具仍可能有自己的内存或崩溃记录策略，因此推荐使用专用、只读、最小权限的 Script。

## 备份数据

快照可能包含人员信息、项目内容、路径和临时签名 URL。请使用 FileVault 或其他加密存储，限制目录权限，不要上传到公共 GitHub 仓库。

## 报告漏洞

公开报告前请移除站点地址、record 内容、API Key、代理凭据和签名 URL。不要在漏洞复现中附带真实快照。
